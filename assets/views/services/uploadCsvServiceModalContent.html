Instructions before upload file

Here is six column in sample file

  1. Service category :: Be sure that category is available before type.
  2. Service name :: Type service name here.
  3. Duration :: Type duration in minutes(only digit). (Example - 0hour 40minute = 40minute,1hour 15minute = 75minute ,2hour 15minute = 135minute)
  4. Buffer Time ::Type buffer time in minutes(only digit). (Example - 0hour 40minute = 40minute,1hour 15minute = 75minute ,2hour 15minute = 135minute)
  5. Sale price :: Type sale price here (In digit)
  6. BTW :: Be sure that BTW is available before type.
